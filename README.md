## Login & Dashboard Project

Description

This project is a simple Login and Dashboard system made using HTML and CSS only.
It shows how page navigation works without using JavaScript or a backend.
All data used in the project is dummy data and for practice purposes only.

## Technologies Used
- HTML
- CSS
- 
## Pages Included
- login.html – Login page
- dashboard.html – Dashboard page
- register.html – Registration page

## How to Run the Project
- Open login.html first.
- Enter any dummy username and dummy password.
- Click the Login button.
- You will be redirected to dashboard.html.
- The dashboard shows:
  - Dummy user information
  - Sample content
    
- Click Logout at the upper right corner
  -  You will be redirected back to the login page
    
- Click Sign Up to go to the registration page
- Fill in dummy information and click Register,
or click Already have an account? to return to the login page

- The Forgot Password option is not functional yet
